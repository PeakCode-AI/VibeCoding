# 对话管理

对话管理用于整理你在 VibeBase 中产生的所有 AI 对话。通过左侧的对话侧边栏，可以新建、搜索、收藏、标记和删除对话。详见 [AI 对话](./chat) 了解对话本身的使用。

## 对话侧边栏

进入对话页面后，左侧为对话侧边栏，自上而下包含：

- **搜索框**：输入关键词按对话标题模糊搜索
- **+ 新建对话**：创建一条新的空对话
- **三个过滤标签**：最近 / 收藏 / 重要
- **对话列表**：按更新时间倒序排列

### 过滤标签

| 标签 | 图标 | 说明 |
| --- | --- | --- |
| 最近 | — | 默认视图，显示全部对话 |
| 收藏 | ⭐ 星标 | 只显示已收藏的对话 |
| 重要 | 🚩 旗帜 | 只显示标记为重要的对话 |

## 新建对话

点击「+ 新建对话」按钮：

1. 系统在后台创建一条新对话记录，返回 `chatId`
2. 页面自动跳转到 `/chat/{chatId}`
3. 你可以在右侧消息区开始输入第一条消息

::: tip 对话即用即建
新建的空对话在发送第一条消息前不会出现在「最近」列表顶部；一旦产生消息，会自动上浮。
:::

## 对话列表项

每个对话在列表中显示：

- 对话标题（取自首条消息摘要，可重命名）
- 最后更新时间，以**相对时间**显示：

| 时间区间 | 显示 |
| --- | --- |
| < 1 分钟 | 刚刚 |
| < 1 小时 | X 分钟前 |
| < 24 小时 | X 小时前 |
| ≥ 24 小时 | X 天前 |

## 选择对话

点击列表中的任意对话：

- 右侧消息区加载该对话的全部历史消息
- 消息按时间正序排列，自动滚动到底部
- 当前选中的对话会高亮显示

## 单条对话操作

将鼠标悬停在某个对话上，右侧会出现 `⋯` 下拉菜单，包含以下操作：

| 操作 | 说明 |
| --- | --- |
| 收藏 | 切换收藏状态（⭐ 图标点亮 / 熄灭） |
| 重要 | 切换重要标记（🚩 图标点亮 / 熄灭） |
| 重命名 | 弹出输入框修改对话标题 |
| 删除 | 删除该对话 |

### 收藏与重要

收藏与重要都是 boolean 标记，可独立设置。常用于：

- 收藏：常用模板对话、效果好的 Prompt
- 重要：需要后续跟进、含关键结论的对话

### 重命名

重命名后，新标题会立即生效，并反映在搜索与列表中。

### 删除

::: danger 删除不可恢复
删除对话会**级联删除**该对话下的所有消息记录，且无法撤销。删除前请确认。
:::
