# 个人设置

个人设置页用于维护你的账户资料，包括头像、工作室信息与联系方式。控制台「系统 → 个人设置」（`/app/console/settings`）。

## 头像上传

页面顶部为头像区域，点击可上传新头像：

- 支持常见图片格式（JPG / PNG 等）
- 单张最大 **5MB**

::: warning 头像依赖 S3 配置
头像上传依赖后端 S3 对象存储。若 S3 未正确配置，上传会返回 **503** 错误。遇到此问题请联系管理员配置 S3，或先使用默认头像。
:::

## 资料表单

下方为资料编辑表单：

| 字段 | 是否可编辑 | 说明 |
| --- | --- | --- |
| 注册类型 | 只读 | 账号注册方式 |
| 用户名 | 只读 | 登录用户名，注册后不可改 |
| 工作室名称 | 可编辑 | 你的工作室 / 团队名称 |
| 团队人数 | 可编辑 | 团队规模 |
| 邮箱 | 可编辑 | 联系邮箱 |

::: info 为什么用户名不可改
用户名是账户的唯一标识，关联订单、积分、对话等全部数据，注册后不支持修改。如需更换用户名，请新建账号并通过 [工单](./ticket) 申请数据迁移。
:::

## 保存修改

1. 修改可编辑字段（工作室名称、团队人数、邮箱等）
2. 点击「保存修改」按钮
3. 系统校验通过后保存成功，页面提示更新完成

::: warning 必填校验
保存时系统会校验必填项。工作室名称与邮箱为必填，留空将无法保存。邮箱格式不正确也会被拦截。
:::

::: tip 邮箱用途
邮箱用于接收账单、安全通知等重要信息，请填写有效地址并保持最新。建议同步在 [安全中心](./security) 开启操作日志监控，及时察觉异常。
:::

## 各字段说明

### 工作室名称

展示在控制台与账单上的工作室 / 团队名称，对外可见，建议使用正式名称。

### 团队人数

用于平台评估你的团队规模，可能影响套餐推荐与服务等级。请如实填写。

### 邮箱

接收平台通知（如 [公告](./announcement)、充值到账、安全告警）的邮箱。修改后即时生效。
