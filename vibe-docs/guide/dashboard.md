# 控制台概览

登录 VibeBase 后，首先进入的就是控制台概览页（`/app/console/dashboard`）。它是你了解账户全貌的入口。

## 进入控制台

1. 访问 VibeBase 首页
2. 点击右上角「登录」，输入用户名密码
3. 登录成功后自动跳转到 `/app/console/dashboard`

::: tip 首次注册
首次注册的用户会自动登录并跳转到设置密码页，设置完成后再进入控制台。
:::

## 页面布局

控制台采用**左侧导航 + 右侧内容**的经典布局：

```
┌──────────┬───────────────────────────────────┐
│          │                                   │
│  侧边栏   │         主内容区                   │
│          │                                   │
│ 首页      │  ┌──────────┐ ┌──────────┐       │
│  概览     │  │ 账户余额  │ │ 今日消耗  │       │
│          │  └──────────┘ └──────────┘       │
│ 财务      │                                   │
│  消费记录 │  ┌─────────────────────────┐     │
│  充值记录 │  │     最近交易              │     │
│  充值套餐 │  │  时间 | 类型 | 积分 | 状态 │     │
│          │  └─────────────────────────┘     │
│ 开发      │                                   │
│  API Key  │     [立即充值]                    │
│  能力列表 │                                   │
│ ...      │                                   │
└──────────┴───────────────────────────────────┘
```

左侧导航分为 5 个分组：首页、财务、开发、分析、系统。详见各功能指南页。

## 概览页内容

### 账户余额卡片

显示当前可用积分余额，以及冻结金额（如有）。

::: info 数据来源
余额来自 `point_accounts.remaining_points`，通过 `GET /api/v1/console/dashboard` 获取。
:::

### 今日消耗卡片

显示今日已消耗的积分总量。

### 最近交易

合并展示最近的消费与充值记录（各取最近 5 条，按时间倒序合并）：

| 列 | 说明 |
| --- | --- |
| 时间 | 交易发生时间 |
| 业务类型 | 对话 / 图像 / 充值等 |
| 积分变动 | `+` 表示充值获得，`-` 表示消费 |
| 状态 | 成功 / 处理中 / 失败 |

### 立即充值按钮

点击跳转到 [充值套餐](./recharge) 页面。

移动端侧边栏会收起，点击左上角汉堡菜单展开。
:::

## 用户菜单

侧边栏底部显示当前登录用户名，点击旁边的退出图标可登出（会撤销当前 Token）。
