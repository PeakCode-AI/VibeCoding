# 积分中心

积分是 VibeBase 的统一计费单位，调用 AI 对话、图像、语音等能力都会消耗积分。本页讲解如何在控制台查看积分余额、明细与消费记录。

## 积分概览

进入控制台「财务 → 积分中心」，顶部展示积分总览：

- **可用积分**：当前账户剩余可用的积分总量（`remaining`）

下方为**积分构成明细**，展示可用积分由哪些来源组成：

| 来源 | 数量 | 说明 |
| --- | --- | --- |
| 免费积分 | 2000 | 注册即赠送 |
| 积分包 | 不等 | 通过 [充值套餐](./recharge) 购买 |
| 会员积分 | 不等 | 开通会员获得 |
| 每日奖励积分 | 2000 | 每日 00:00 重置，剩余 1000 上限 |

::: warning 每日积分每日清零
每日奖励积分每天 00:00 重置为 2000，前一天未用完的部分（最多累计 1000）会保留并叠加。建议优先消耗每日积分。
:::

## 积分消耗顺序

当一次调用需要扣费时，系统按以下顺序消耗积分：

```
每日积分 → 会员积分 → 积分包 → 免费积分
```

::: info 为什么是这个顺序
每日积分会过期，优先消耗可避免浪费；免费积分永不过期，放在最后兜底。
:::

## 两个标签页

### 积分明细

展示所有积分变动记录，按时间倒序排列：

| 列 | 说明 |
| --- | --- |
| 详情 | 变动原因（如「AI 对话消耗」「充值到账」） |
| 日期 | 发生时间 |
| 积分变更 | `+` 表示增加，`-` 表示消耗 |

底部显示**累计消耗汇总**，并支持分页浏览。

### 充值记录

展示所有充值订单，与 [充值记录](./recharge) 页面数据一致。

## 积分消费记录页

控制台「财务 → 积分消费记录」（`/app/console/points-consume`）提供更细粒度的消费查询，支持以下筛选条件：

| 筛选项 | 说明 |
| --- | --- |
| 用户名 | 按子账号用户名过滤 |
| 流水号 | 按交易流水号精确查找 |
| 业务类型 | 对话 / 图像 / 语音 / 文本 |
| 日期范围 | 起止日期 |

查询结果支持 **CSV 导出**，便于对账与统计。

::: tip 子账号消费
主账号可在此查看所有子账号的消费明细。子账号概念见 [子账号管理](./account)。
:::
