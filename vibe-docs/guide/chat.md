# AI 对话

AI 对话是 VibeBase 的核心功能。本页教你如何使用普通对话、Agent 模式、联网搜索、深度研究、深度思考等全部对话能力。

## 进入对话

控制台左侧导航「首页」下可进入对话。对话界面由三部分组成：

```
┌────────────┬─────────────────────────┬─────────────┐
│            │                         │             │
│ 对话列表    │      消息区              │  研究面板    │
│            │                         │  (可选)      │
│ 搜索框      │  ┌───────────────────┐  │             │
│ + 新建对话  │  │ AI: 你好！         │  │  研究计划    │
│            │  └───────────────────┘  │  步骤进度    │
│ 最近/收藏/  │  ┌───────────────────┐  │  来源列表    │
│ 重要       │  │ 你: 帮我分析...     │  │             │
│            │  └───────────────────┘  │             │
│ 对话1      │                         │             │
│ 对话2      │                         │             │
│            │  ┌───────────────────┐  │             │
│            │  │ [输入框]          │  │             │
│            │  │ 模型|搜索|思考|研究 │  │             │
│            │  └───────────────────┘  │             │
└────────────┴─────────────────────────┴─────────────┘
```

## 两种对话模式

在输入框左侧的下拉菜单可切换模式：

### 普通对话（默认）

日常通用对话。适合问答、写作、翻译、解释等。

- 可手动开关：联网搜索、深度思考、深度研究
- 响应速度较快

### Agent 模式

复杂任务自动化处理。切换到 Agent 模式会**自动开启**联网搜索、深度研究、深度思考。

- 适合数据分析、报告撰写、深度调研
- 会展示任务类别徽章（数据分析 / 表格 / 网页 / 文档 / 幻灯片）
- 响应较慢但更完整

::: tip 模式选择建议
- 简单问答 → 普通对话
- 需要最新信息 → 普通对话 + 联网搜索
- 复杂分析任务 → Agent 模式
:::

## 发送消息

1. 在底部输入框输入消息（上限 4000 字符）
2. 按 `Enter` 或点击发送按钮
3. AI 回复会**逐字流式**显示

::: info 积分校验
发送前，系统会预估本次对话的 token 消耗（输入长度 × 2）。如果余额不足，发送会被拦截，提示你去充值。
:::

## 联网搜索

点击输入框上方的🌐（地球）图标开启联网搜索。

开启后：

1. AI 会先显示「正在全网搜索...」
2. 搜索结果以结构化卡片展示（带网站图标）
3. 点击「查看详情」可在右侧搜索面板展开来源
4. AI 基于搜索结果生成回答

## 深度思考

点击⚛️（原子）图标开启深度思考。

开启后：

- AI 回复前会展示**可折叠的推理过程**（「正在思考...」/「已思考」）
- 推理过程支持结构化多 Agent 思考链展示
- 点击可展开 / 收起查看推理细节

## 深度研究

点击🔭（望远镜）图标开启深度研究。

::: tip 深度研究是重功能
深度研究会启动多 Agent 编排（协调者 / 规划者 / 研究员 / 报告者），耗时较长（约 15 分钟），消耗积分较多。适合需要深度调研的场景。
:::

开启后的流程：

### 1. 计划确认

AI 会先生成一份研究计划，以卡片形式插入对话：

- 显示研究标题与步骤
- 你可点击「确认」开始、「取消」终止、「修改」调整

### 2. 澄清问答（可选）

如果 AI 需要更多信息，会插入一张澄清卡片：

- 支持单选 / 多选 / 文本回答
- 可「确认」提交、「跳过」、「取消」

### 3. 执行研究

右侧会打开「研究面板」，实时显示：

- 研究步骤列表（待处理 / 进行中 / 已完成 / 错误）
- 每步的图标（计划 / 搜索 / 浏览 / 思考 / 撰写）
- 进度百分比
- 预计剩余时间
- 信息来源列表

### 4. 下载报告

研究完成后，AI 会生成一份完整报告，可点击「下载」保存为 Markdown 文件。

## 图像理解

点击输入框上方的📎（回形针）图标可上传图片：

- 支持 JPG / PNG 等常见格式
- 单张最大 10MB
- 上传后图片会内联显示在消息中
- 可针对图片提问（如「描述这张图」「图中的数据说明了什么」）

::: info 需配置视觉模型
图像理解需要后端配置 `VL_*` 视觉模型。未配置时会返回降级演示文案。详见 [LLM 配置](../configuration/llm)。
:::

## 选择模型

点击输入框上方的模型名可打开模型选择器：

- 搜索 / 选择可用模型
- 不同模型在能力、速度、价格上有差异

::: warning 当前模型列表
模型列表目前为静态配置（GPT-4 / GPT-3.5 Turbo / DeepSeek），默认选择首个 DeepSeek 模型。实际可用模型取决于后端 LLM 配置。
:::

## Prompt 库

在输入框中输入 `/` 可唤出 Prompt 库选择器：

- 搜索 / 过滤 Prompt
- 方向键导航，Enter 插入
- 分「我的」与「广场」

## 消息操作

每条 AI 回复下方有操作按钮：

| 按钮 | 功能 |
| --- | --- |
| 📋 复制 | 复制回复内容到剪贴板 |
| 👍 喜欢 | 点赞（反馈质量好） |
| 👎 反对 | 踩（反馈质量差） |

点赞 / 反对数据用于 AI 质量评估。

## 停止回答

AI 正在流式回复时，发送按钮会变为「停止回答」按钮，点击可中断当前生成（基于 AbortController）。

## 积分扣除

对话过程中，积分扣除会通过 SSE 实时通知：

- `points_deducted` — 已扣除 N 积分
- `points_warning` — 余额较低预警
- `points_insufficient` — 余额不足

余额变化会实时反映在界面上。

::: tip 谁不扣积分
管理员账号（user_id = 1）与开发账号（dev_001）对话不扣积分，方便测试与演示。
:::
