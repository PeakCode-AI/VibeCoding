# 充值套餐

当积分余额不足时，可以通过购买充值套餐快速补充积分。本页讲解如何选购套餐、查看订单与导出充值记录。

## 进入充值页

控制台「财务 → 充值套餐」（`/app/console/recharge-package`），页面顶部显示当前可用余额，下方为三档套餐卡片。

## 套餐对比

| 套餐 | 价格 | 积分 | 适合 |
| --- | --- | --- | --- |
| 基础版 basic | ¥99 | 10,000 | 个人轻度使用 |
| 专业版 pro（推荐） | ¥399 | 50,000 | 团队日常使用 |
| 企业版 enterprise | ¥999 | 200,000 | 企业批量采购 |

每张卡片会列出包含的特性（如调用上限、并发数、优先支持等），「专业版」带有 **推荐** 徽章。

::: warning 套餐价格说明
以上套餐与价格为**演示数据**，用于展示控制台功能，与官网首页源码档位（source-code tier）的定价不同。正式上线价格以官网为准。
:::

## 购买流程

1. 选择目标套餐卡片
2. 在支付方式中选择：**微信支付 / 支付宝 / 银行转账**
3. 点击「立即购买」
4. 系统调用 `createRechargeOrderAPI` 创建订单，返回 `order_id` 与 `payment_url`
5. 跳转到支付页完成付款
6. 支付成功后积分自动到账

::: info 订单状态
订单创建后处于「待支付」状态，支付回调确认后变为「成功」。若长时间未到账，可在 [充值记录](#充值记录页) 查询订单状态。
:::

## 充值记录页

控制台「财务 → 充值记录」（`/app/console/recharge-record`）展示所有充值订单，支持以下筛选：

| 筛选项 | 说明 |
| --- | --- |
| 交易号 | 按订单流水号精确查找 |
| 开始日期 | 订单创建起始时间 |
| 结束日期 | 订单创建截止时间 |
| 状态 | 成功 / 处理中 / 失败 |

订单列表包含：交易号、套餐、金额、积分、支付方式、状态、创建时间。

### 分页与导出

- 列表底部支持分页浏览
- 点击「导出 CSV」可将当前筛选结果下载为电子表格，方便财务对账

::: tip 企业开票
银行转账的企业订单如需发票，请通过 [工单系统](./ticket) 提交开票申请，附上订单号。
:::
