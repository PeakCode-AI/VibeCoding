# 公告中心

公告中心发布 VibeBase 平台的官方通知，包括维护计划、新功能上线、定价调整、版本更新与安全通告。帮助你在第一时间了解平台动态。

## 页面布局

公告中心采用**时间线**布局，按发布时间倒序排列：

- 每条公告显示类型徽章、发布日期、标题与正文
- 最新一条公告带有 **最新** 标记
- 时间线样式便于按时间顺序浏览历史公告

## 公告类型

每条公告都有一个类型徽章，对应 5 种公告类别：

| 类型 | 徽章 | 含义 |
| --- | --- | --- |
| 维护 | 维护 | 计划内的系统维护、停机通知 |
| 新功能 | 新功能 | 新增能力或功能上线 |
| 定价 | 定价 | 积分价格、套餐调整 |
| 版本 | 版本 | 平台版本发布说明 |
| 安全 | 安全 | 安全通告、漏洞修复 |

::: tip 关注「安全」与「维护」类公告
安全通告可能要求你更新 API Key 或检查配置；维护公告会告知停机时间，请提前安排业务。
:::

## 阅读公告

每条公告卡片包含：

- **类型徽章**：颜色区分公告类别
- **发布日期**：公告发布时间
- **标题**：一句话概括
- **正文**：详细说明，可能包含操作指引或时间安排

## 如何响应

不同类型公告的常见处理方式：

| 类型 | 建议操作 |
| --- | --- |
| 维护 | 记录维护时间，提前降低流量或设置降级 |
| 新功能 | 评估是否应用到自己的业务 |
| 定价 | 关注 [积分中心](./points) 与 [充值套餐](./recharge) |
| 版本 | 了解版本变更，检查兼容性 |
| 安全 | 按指引 [更新密码](./security) 或轮换 [API Key](./apikey) |

::: tip 定价类公告与官网
定价调整公告中的套餐信息应与官网首页（Landing page）展示的档位一致。若发现控制台套餐与官网定价不符，以官网为准，并可提交 [工单](./ticket) 反馈。
:::

::: info 公告来源
公告由 VibeBase 运营团队通过后台统一发布，所有用户均可见。如对某条公告有疑问，可提交 [工单](./ticket) 咨询。
:::

## 浏览建议

- 定期查看公告中心，避免遗漏重要通知
- 「维护」类公告发布后，请提前安排业务流量
- 历史公告会保留在时间线中，可随时回溯查阅
