# ICP 备案

如果你的服务器部署在**中国大陆**，域名必须完成 ICP 备案才能通过 80 / 443 端口对外提供网站服务。本页是备案流程指引。

## 什么是 ICP 备案

ICP 备案（Internet Content Provider）是中国工信部对境内提供非经营性互联网信息服务的准入制度。简单说：**大陆服务器 + 域名建站 = 必须备案**。

::: warning 不备案的后果
- 运营商会在网络层拦截未备案域名的 80 / 443 请求
- 云服务商不会为你开放 web 服务
- 无法正常对外访问
:::

## 哪些情况可以不备案

| 情况 | 是否需要备案 |
| --- | --- |
| 服务器在中国大陆 | ✅ 必须 |
| 服务器在港澳台 | ❌ 不需要 |
| 服务器在海外（如 AWS 东京、Cloudflare） | ❌ 不需要 |
| 仅用 IP 访问（不用域名） | ⚠️ 灰色地带，运营商可能限制 |
| 只做 API（非网站） | ⚠️ 建议仍备案 |

::: tip 免备案方案
如果不想走备案流程：
- 选择海外服务器（香港 / 新加坡 / 日本等）
- 用 Cloudflare 等 CDN 代理（源站在海外）
- 缺点：国内访问延迟较高
:::

## 备案流程（以阿里云为例）

### 1. 准备材料

| 材料 | 说明 |
| --- | --- |
| 身份证 | 个人备案需本人身份证 |
| 域名证书 | 域名注册成功后的所有权证明 |
| 服务器 | 需购买大陆云服务器（至少 3 个月） |
| 手机号 | 用于接收验证码与审核电话 |
| （企业）营业执照 | 企业备案需提供 |

### 2. 提交备案

1. 登录云服务商备案系统（如 [阿里云备案](https://beian.aliyun.com)、[腾讯云备案](https://cloud.tencent.com/product/ba)）
2. 选择「新增备案」
3. 填写主体信息（个人 / 企业）
4. 填写网站信息（域名、网站名、服务内容）
5. 上传材料
6. 完成核验（人脸识别 / 幕布拍照）

### 3. 等待审核

| 阶段 | 耗时 | 说明 |
| --- | --- | --- |
| 云商初审 | 1-2 工作日 | 云服务商审核材料 |
| 短信核验 | 即时 | 收到工信部短信，24h 内验证 |
| 管局审核 | 5-20 工作日 | 各省通信管理局审核 |
| 总计 | **7-20 工作日** | 视省份而定 |

::: tip 提前规划
备案期间域名无法通过 80/443 访问。建议在产品开发期就启动备案，避免上线时被卡。
:::

### 4. 备案成功

审核通过后：

- 获得 ICP 备案号（如「沪ICP备XXXXXXXX号」）
- 需在网站底部展示备案号，并链接到 [工信部查询页](https://beian.miit.gov.cn)
- 域名可正常通过 80/443 访问

## 在 VibeBase 中展示备案号

按法规要求，网站底部必须展示备案号。你可以在前端 Footer 组件中添加：

```vue
<!-- src/components/home/MainFooter.vue 或对应位置 -->
<template>
  <footer>
    <!-- 其他内容 -->
    <a href="https://beian.miit.gov.cn" target="_blank" rel="noopener">
      沪ICP备XXXXXXXX号-X
    </a>
  </footer>
</template>
```

::: tip 公安备案
部分省份还要求「公安联网备案」（网安备案）。如果当地有要求，需额外到 [全国互联网安全管理平台](http://www.beian.gov.cn) 提交。
:::

## 备案后的合规要点

::: warning 备案后注意事项
- **备案信息变更要及时更新** — 换主体、换域名、换服务器都要变更备案
- **不要超范围经营** — 个人备案不能做经营性服务（如电商），经营性需办 ICP 经营许可证（EDI）
- **保留访问日志** — 按要求留存日志至少 60 天
- **内容合规** — 不得包含违法违规内容
:::

## VibeBase 的备案定位

VibeBase 作为 AI 产品平台，备案时「服务内容」可填：

- 个人备案：「个人技术博客 / 学习分享」（注意：个人备案通常不允许明显的经营行为）
- 企业备案：「软件和信息技术服务业」（可覆盖 SaaS、AI 服务）

::: danger 经营性服务需额外许可
如果你的 AI 产品面向公众收费（充值套餐、订阅），属于经营性互联网信息服务，除 ICP 备案外可能还需要：
- **ICP 经营许可证**（增值电信业务经营许可证）
- 若涉及 AI 生成内容，还需关注《生成式人工智能服务管理暂行办法》的算法备案

具体要求请咨询当地通信管理局或法律顾问。
:::
